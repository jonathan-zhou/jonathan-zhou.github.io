What are the best practices associated with using classes vs. ids?

What are classes and IDs? They are CSS elements that can apply and manipulate HTML elements, similar to an image manipulatator like Photoshop how you can adjust image length, size, position, etc.

The difference between the two is that Classes can be applied to multiple HTML elements which can make things simplier if you need something to show up on every page such as a sidebar while IDs should only be used once in a single HTML element, in other words it should be unique to once HTML element.

Both selectors also have their own symbol to identify whether they are a Class or ID. Classes use the "." (fullstop) symbol whilst IDs use a # (hash) symbol.
